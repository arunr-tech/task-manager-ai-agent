# Task Manager AI Agent

## Overview

Task Manager AI Agent is an interactive Python-based AI agent that helps users manage tasks efficiently.

## Features

- Add tasks
- Complete tasks
- Remove tasks
- Show pending tasks
- Show completed tasks
- Track progress
- Recommend the next task
- Interactive menu system

## Technologies Used

- Python
- Dictionaries
- Lists
- Functions
- Loops
- Conditional statements

## How to Run

```bash
python main.py
```

## Menu

1. Show Tasks
2. Add Task
3. Complete Task
4. Remove Task
5. Show Progress
6. Recommend Task
7. Exit
